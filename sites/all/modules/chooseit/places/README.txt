INSTALLATION STEPS FOR PLACES MODULE
------------------------------------------------------------
    -Activate the module. (Admin -> modules -> places).
    -(The database table: places) will be created for you.
    -Manage the permissions. (Admin -> people -> permissions).
    -The implementation of the module:
     Admin -> Structure -> menus > (choose the menu you want) -> insert new link -> insert name -> insert (places) as a link path -> and choose the parent menu -> save -> clear cache -> done!

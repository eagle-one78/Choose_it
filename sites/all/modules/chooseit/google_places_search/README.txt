INSTALLATION STEPS FOR GOOGLE PLACES SEARCH MODULE
------------------------------------------------------------
    -Activate the module. (Admin -> modules -> google places search).
    -Manage the permissions. (Admin -> people -> permissions).
    -Insert your Google App code in google_autocomplete.html (google_places_search -> google_autocomplete.html).
    -The implementation of the module:
     Admin -> Structure -> menus > (choose the menu you want) -> insert new link -> insert name -> insert (google_places_search) as a link path -> and choose the parent menu -> save -> clear cache -> done!
